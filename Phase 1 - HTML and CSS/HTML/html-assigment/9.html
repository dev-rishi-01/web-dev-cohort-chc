Format Type Format Description
Video MP4 (H.264) Most widely supported video format; good quality with small file size; works on all major browsers.
Video WebM Open-source format designed for the web; efficient compression; supported by modern browsers.
Video OGV (Ogg Video) Open, royalty-free video format; supported by many HTML5 video players.
Audio MP3 Most popular audio format; excellent compression and universal browser support.
Audio OGG Vorbis Open-source, high-quality audio format and a common alternative to MP3.
Audio WAV Uncompressed, high-quality audio; large file sizes but widely supported on the web.
